## 설치

### macOS

1. `.dmg` 를 열고 **Keural** 을 응용 프로그램 폴더로 드래그
2. **"손상되었기 때문에 열 수 없습니다"** 가 뜨면:
   ```bash
   xattr -cr /Applications/Keural.app
   ```
3. 또는 차단되면 **시스템 설정 → 개인정보 보호 및 보안 → "확인 없이 열기"**

### Windows

1. `.exe` 실행
2. SmartScreen 경고 시 **추가 정보 → 실행**

### Linux

```bash
chmod +x Keural-linux-x86_64.AppImage
./Keural-linux-x86_64.AppImage
```
